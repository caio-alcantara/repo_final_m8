Sobre o curso:
O programa executivo
Data Science for Decision Makers
foi desenvolvido com o objetivo de apresentar técnicas de análise e investigação de dados para tomada de decisão, com aplicação para profissionais de mercado de diversos setores.
Neste programa, os alunos aprenderão sobre investigação de problemas e escolha dos melhores dados e modelos para iniciar um projeto, técnicas para análise descritivas de dados, modelos e frameworks para análise preditiva baseada em dados, além de testes de hipóteses e design de experimentos para obter
insights
importantes para a tomada de decisão.
Os conteúdos são abordados utilizando cases de universidades internacionais renomadas, e mediados por professores com vasta experiência no mercado.
Tópicos abordados no curso:
Módulo 1 – Diagnóstico: Definindo o Problema
Módulo 2 – Análise Descritiva: Investigando o Passado
Módulo 3 – Análise Preditiva: Inferindo o Futuro
Módulo 4 – Experimentação: Testando Mudanças
Ao final do curso, você será capaz de:
Aplicar heurísticas e tratar possíveis vieses de dados no processo de tomada de decisão
Resolver problemas com análise exploratória de dados
Estruturar observações de dados com técnicas de visualização
Aplicar modelos preditivos para realizar inferências sobre possíveis resultados futuros
Desenvolver experimentos para validar hipóteses
